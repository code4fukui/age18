# 18際意識調査

このプロジェクトは、日本財団が実施した「18歳意識調査」のデータを可視化するウェブアプリです。9カ国の18歳の意識を調査した結果を、棒グラフやパイチャートで示しています。

## デモ
https://code4fukui.github.io/age18/

## 機能
- 国や性別を選択して、18歳の意識を可視化
- 項目別にグラフを表示

## データ・API
「[日本財団「18歳意識調査」第20回 テーマ：「国や社会に対する意識」（9カ国調査） | 日本財団](https://www.nippon-foundation.or.jp/who/news/pr/2019/20191130-38555.html)」のデータを加工して使用しています。
